# Drug Database Web App

A full-stack web application built using Java and Spring Boot that allows users to manage and search a database of pharmaceutical drugs.

## Features
- Add new drugs to the database
- View all stored drugs
- Search for specific drugs
- Edit and delete entries
- Simple UI using Thymeleaf

## Tech Stack
- Java
- Spring Boot
- Spring MVC
- Thymeleaf
- H2 / MySQL
- Maven

## How to Run
1. Clone the repository
2. Open in IntelliJ
3. Run the main application class
4. Visit `http://localhost:8080`
